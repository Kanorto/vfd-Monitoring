# vfd-Monitoring
Это система мониторинга для дисплеев покупателей, которые основаны на VFD дисплеях. Данная программа изначально разработана для VFD DISPLAY PD-2800 от ATOL.

## Автообновления
- Приложение теперь умеет проверять GitHub Releases, скачивать новый релиз и применять его при следующем запуске или во время выхода из трея.
- По умолчанию используется репозиторий `Kanorto/vfd-Monitoring`, но при необходимости `github_repo` можно переопределить в `vfd_config.json` в формате `owner/repo` или ссылкой на GitHub-репозиторий.
- После установки новой версии changelog сохраняется в конфиг и доступен из трея через пункт `Показать changelog`.
- Если автообновление несколько раз подряд не сможет скачать релиз или проверить его, приложение покажет alert со ссылкой на страницу релиза для ручного обновления.
- На стартовом экране вторая строка теперь показывает текущую версию программы.
- Автообновление в первую очередь выбирает любой `.exe`-файл из релиза GitHub, даже если в релизе также лежит файл с именем текущего `.py`; другие типы файлов используются только как запасной вариант.

## Пользовательское форматирование
- В `vfd_config.json` добавлены разделы `metric_formats` и `line_spacing`.
- `metric_formats` управляет шаблонами строк для `cpu`, `gpu`, `ram`, `disk` и `network`.
- Для CPU/GPU доступны плейсхолдеры `{usage}`, `{temp}`, `{full_prefix}`, `{short_prefix}`, `{degree}`.
- Для RAM доступен `{value}`.
- Для диска доступны `{read}` и `{write}`, для сети — `{recv}` и `{send}`.
- `line_spacing.primary` и `line_spacing.secondary` задают основной отступ между блоками на первой и второй строках.
- `line_spacing.primary_compact` и `line_spacing.secondary_compact` задают отступ, который используется, когда строка уже не помещается и приложение переходит в компактный режим.
- Быстро открыть конфиг можно из трея через пункт `Открыть конфиг`.

## Горячие клавиши
- В трее появился раздел `Горячие клавиши`.
- Можно полностью включать или отключать глобальные хоткеи через настройку `hotkeys_enabled`.
- Настройки самих комбинаций хранятся в `hotkeys` внутри `vfd_config.json`.
- Окно `Настроить клавиши...` умеет записывать комбинацию в момент нажатия клавиш и сохранять её в конфиг.
- По умолчанию доступны:
  - `Ctrl+Alt+D` — включение/выключение дисплея;
  - `Ctrl+Alt+U` — ручная проверка обновлений;
  - `Ctrl+Alt+C` — открытие changelog.
